# E-Shop


## Project description

Simple online store, no link to the backend yet.

Embrace `React` functional components, `Hooks`.

### install & run

download dependencies

- `npm install`

run local server

- `npm run dev`

### technology stack

- `React 17.0.2`

- `Vite` scaffold

- `CSS Module`
